# Hangman Game 🎮

## 📌 Description
This is a simple Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

The game randomly selects a word, and the player has to guess it one letter at a time. The player has 6 attempts to guess the word correctly.

## ✨ Features
- Random word selection
- Letter-by-letter guessing
- Tracks guessed letters
- Prevents duplicate guesses
- Displays remaining attempts
- Win and Game Over messages

## 🛠️ Technologies Used
- Python 3
- Random Module

## ▶️ How to Run
1. Download or clone this repository.
2. Open the project in VS Code.
3. Run the `hangman.py` file.
4. Start guessing the letters!
